## Install, import and basic usage

For instalation:
```bash
npm i als-simple-css
```

Use in browser:
```html
<script src="/node_modules/als-simple-css/simple.js"></script>
```

Use as commonjs:
```js
const Simple = require('als-simple-css')
```

Use as module:
```js
import Simple from 'als-simple-css/simple.mjs'
```
